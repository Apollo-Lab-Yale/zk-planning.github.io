1Yale University

Abstract

We present an approach that addresses the Zero-Knowledge task planning problem, i.e. planning high-level actions to achieve a given goal without any task-specific information. Our approach leverages a large language model (LLM) to decompose natural language instructions into subtasks and generate behavior trees (BTs) for execution. If errors arise during task execution, the approach also uses an LLM to adjust the BTs on-the-fly in a refinement loop. Experimental validation in the AI2-THOR simulator demonstrates our approach's effectiveness in improving overall task performance compared to alternative approaches, even though all of these approaches rely on task-specific knowledge. Our work demonstrates the potential of LLMs to effectively address the Zero-Knowledge task planning problem, providing a robust framework for automated behavior generation in dynamic environments.
